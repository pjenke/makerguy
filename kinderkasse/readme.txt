*** HARDWARE ***
- Raspberry Pi 2, model B (should work with all versions)
- Raspberry Pi camera Rev 1.3 (should work with all versions) -> Adjust focus to "close": https://www.raspberrypi.org/blog/macro-photography-with-the-camera-board/
- LCD Display 2004, I2C, 5V with LCMI1602 board (needs to be connected via 2-channel logic converter 3.3V <-> 5V)
- Mini USB speakers
